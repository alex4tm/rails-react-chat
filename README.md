## Description

This is a slack-like-chat that is buit on Rails, React, authentication using devise gem, PostgreSQL, Redis and action cable for real-time messages.

#### Technologies

- Ruby on Rails
- Javascript
- ReactJS
- Redis

## How To Use

#### Installation

Fork or download the repository.

```html
    npm install
```

Create the database:
```html
    rake db:create
```

Run server:
```html
    rails s
```
